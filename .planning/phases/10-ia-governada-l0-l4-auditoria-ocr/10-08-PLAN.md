## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser upload → /api/ocr | file leaves the browser to the route; route enforces auth + MIME + size + ZDR (Plan 05) |
| extracted fields → patient commit | model output untrusted until human confirms; reviewer edits are authoritative |
| client → confirm/reject actions | assertNotReadOnly + tenant scope enforced server-side (Plan 05) |

| T-10-34 | Information Disclosure | raw CPF/RG rendered or logged client-side | mitigate | only the 4 pilot fields shown; no file bytes echoed; route already masks logs (Plan 05) |
