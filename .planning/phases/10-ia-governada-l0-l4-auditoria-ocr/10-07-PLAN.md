|----------|-------------|
| browser → /conformidade routes | proxy gates by conformidade module role; pages re-check role server-side |
| client approve/reject → Server Action | alçada + assertNotReadOnly enforced server-side; UI disable is cosmetic only |
| audit row payload → DOM | old_values/new_values + approval payload may contain PII; render guards mask sensitive fields |
