| segment query → LLM | Only first name crosses to the model |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-18-15 | Tampering (consent boundary) | send to non-consented patient (LGPD) | mitigate | Segment query joins patient_consents (marketing_whatsapp, revoked_at IS NULL); re-applied at dispatch time. |

| T-18-17 | Repudiation | who approved a blast | mitigate | approveRequest writes approver + decided_at + logBusinessEvent; approval_request_id stored on campaign. |
