Helper interno selectFifoBatch (comentar `// FIFO`): CAS guard atômico (Pitfall 2):
```typescript
// UPDATE product_batches SET saldo_disponivel = saldo_disponivel - qtd
//   WHERE id = (SELECT id FROM product_batches WHERE product_id AND unit_id AND saldo_disponivel > 0
//     AND deleted_at IS NULL ORDER BY created_at ASC LIMIT 1)
//   AND saldo_disponivel >= qtd RETURNING id
```
Se retornar vazio (consumido por concorrente OU saldo insuficiente), tentar próximo lote; se nenhum lote → batch_id = null (D-09 saldo negativo permitido). Como Supabase JS não faz subquery inline, implementar: SELECT o lote FIFO → UPDATE ...WHERE id AND saldo_disponivel >= qtd RETURNING id → se vazio, loop próximo lote.

drawMaterialsForProcedures(appointmentId, clinicId, actorId): usar createAdminClient (cruza appointment→templates→batches sem RLS de sessão).
1. Resolver unit_id: SELECT appointments.unit_id WHERE id=appointmentId (Open Question 3 — appointment_procedures não tem unit_id).
2. SELECT appointment_procedures WHERE appointment_id=appointmentId (id, service_id).
3. Para cada procedure: SELECT service_material_templates WHERE service_id AND deleted_at IS NULL (product_id, qtd_padrao).
4. Para cada template: selectFifoBatch(product_id, unit_id, qtd_padrao). Ler custo médio snapshot (products.custo_medio OU custo_unitario do lote). Insert stock_draws { clinic_id, unit_id, product_id, batch_id, appointment_procedure_id, qtd, custo_unitario_snapshot, tipo: 'automatico', created_by: actorId }.
5. Recalcular saldo do produto na unidade (SUM batches). Se saldo <= estoque_minimo → runStockReplenishmentAgent({...}). Se saldo < 0 → insertStockAlert(...,'minimo') (marca negativo).
6. logBusinessEvent action 'stock.draw.auto'.
Toda a função em try/catch interno por template — falha em um material NÃO aborta os demais nem o atendimento (D-09).

createManualDraw(input): 'use server' export. getActor + gate WRITER_ROLES. `stockDrawSchema.parse(input)` (motivo obrigatório D-19). selectFifoBatch para debitar lote. Disparar check de mínimo (agente) igual à automática. revalidatePath('/clinica/estoque/produtos').
