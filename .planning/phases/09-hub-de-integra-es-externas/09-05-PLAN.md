      4. `logBusinessEvent({ tenantId, actorId, action:'connector.reprocess', details:{ connectorId, requeued } })` — NO payloads, NO secret.
      5. Return `{ success:true, requeued: ids.length }`.
  All exports MUST be async functions or `type`/`interface` (Next.js 'use server' rule — D-127).
