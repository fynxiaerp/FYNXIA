## Spacing Scale

All values are multiples of 4. Tailwind v4 with CSS variables — use Tailwind utility classes, not raw pixels.

| Token | Value | Tailwind class | Usage |
|-------|-------|----------------|-------|
| xs | 4px | `gap-1` / `p-1` | Icon gaps, inline icon-to-text padding |
| sm | 8px | `gap-2` / `p-2` | Badge padding, compact row gaps |
| md | 16px | `gap-4` / `p-4` | Default element spacing, form field gaps |
| lg | 24px | `gap-6` / `p-6` | Section padding (page `main` uses `p-6`) |
| xl | 32px | `gap-8` / `p-8` | Layout gaps between major sections |
| 2xl | 48px | `py-12` | Empty-state vertical padding |
| 3xl | 64px | `py-16` | Full-page empty-state vertical centering |

**Exceptions:**
- Touch-target minimum on icon-only action buttons: `size-8` (32px) or `size-9` (36px) via shadcn Button size props — never below 32px.
- KPI card minimum height: `min-h-[72px]` — matches `CashFlowTotals` established pattern.
- PageHeader fixed height: `h-16` (64px) — matches `PageHeader` component.
- Table row padding: `px-4 py-3` for card headers, standard `TableCell` padding for rows — matches prototype pattern.

### Accent Reserved For (explicit list)
Accent (`--primary` / `--ring`) is applied **only** to:
1. Primary CTA buttons (`Button` default variant)
2. Active sidebar nav item indicator
3. Focus rings (`ring-ring`) on interactive elements
4. Active/selected state on tab underline
5. `hover:text-primary` on financeiro hub nav card hover
6. `hover:border-primary` on financeiro hub nav card hover
7. The `ring` token on NFS-e / lote status progress indicators (spinner border)

Accent is **never** used on: table row backgrounds, icon fill in muted states, empty-state icons, secondary info text.

**Page title:** "Ordens de Serviço"
**Breadcrumbs:** Financeiro > Faturamento > OS

## Interaction Patterns

### State Machine Buttons (OS lifecycle — D-10)

OS `rascunho` state Sheet footer:
```
[Faturar OS]          ← Button default (primary), full-width
[Cancelar OS]         ← Button variant="outline", destructive-labeled, triggers alçada
```

OS `faturada` state Sheet footer:
```
[Cancelar OS]         ← Button variant="outline", destructive-labeled, full-width
```

OS `cancelada` state Sheet footer: empty — read-only notice only.

State machine is enforced server-side. Button rendering is client-side guard only (per D-19 estorno flow).

### Destructive Actions — Confirmation Approach

All 3 destructive actions use a shadcn `AlertDialog` (not `Dialog`) pattern:

1. **Faturar OS** — AlertDialog with confirmation (irreversible without estorno)
2. **Cancelar OS faturada** — AlertDialog + Textarea motivo (triggers Phase 10 alçada flow)
3. **Cancelar NFS-e** — AlertDialog + Textarea motivo (gated by provider cancelamento window)

Pattern: AlertDialog trigger on button click → title + description → textarea (when motivo required) → "Confirmar" (destructive) + "Voltar" (outline).

Never use `window.confirm()`. Never inline-delete without AlertDialog.

### Table Row Actions

Use shadcn `DropdownMenu` (`src/components/ui/dropdown-menu.tsx`) for per-row actions:
- OS table: "Ver detalhes", "Faturar" (se rascunho), "Cancelar" (se faturada)
- NFS-e table: "Ver nota", "Cancelar NFS-e" (se emitida e dentro da janela)
- TISS guides table: "Ver guia", "Registrar recurso" (se glosada)
- Operadoras table: "Editar", "Ver tabela de preços", "Desativar"

### Loading States

All server data is loaded in RSC pages. Pass skeleton fallbacks via `loading.tsx` (`Suspense` boundary pattern matching existing pages):
- KPI row: 4× `Skeleton className="h-[72px] rounded-xl"` in grid
- Table: `Skeleton className="h-64 rounded-xl w-full"`

### URL-Persisted Filters (nuqs)

All list screens use nuqs for filter state. Keys:
- `?month=YYYY-MM` — período (OS, NFS-e, guias, glosas)
- `?status=` — status filter (OS, NFS-e, guias, glosas)
- `?pagador=particular|convenio` — OS pagador filter
- `?operadora=<id>` — convênios/glosas operadora filter

Matches established pattern from `fluxo-de-caixa` (`?month=`, `?unit=`, `?cc=`).

### Monetary Formatting

All monetary values use `formatBRL` from `src/lib/format/money.ts`. KPI display amounts use `text-2xl font-semibold tabular-nums`. Table cell amounts use `text-right font-medium tabular-nums`. Use U+2212 (true minus − ) for negative values — matches `CashFlowTotals` pattern.

### Glosa Rate Threshold Coloring

```tsx
<span className={glosaRate >= 6 ? 'text-destructive' : 'text-muted-foreground'}>
  {pct(glosaRate)}
</span>
```
Source: `convenios/page.tsx` line 108. Rate threshold = 6%.

## Accessibility Contract

- PageHeader `h1` uses `text-xl font-semibold font-display leading-tight` — Screen readers announce page on load.
- KPI cards: each `Card` gets `aria-label="[metric name]: [formatted value]"` — matches `CashFlowTotals` pattern.
