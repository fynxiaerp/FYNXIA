## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser form → clinical-documents actions | untrusted input; Zod-validated + admin/dentist-gated + tenant-scoped in the action (Plan 04) |
| read-only role → issue/sign CTA | x-read-only header hides CTAs; assertNotReadOnly blocks at the action layer (defense in depth) |
| allergy alert display | reasons are non-identifying strings from the server; no patient PII in the alert payload |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
