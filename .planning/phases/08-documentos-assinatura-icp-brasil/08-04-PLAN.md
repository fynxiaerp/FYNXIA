| Boundary | Description |
|----------|-------------|
| client → template action | untrusted name/category/content crosses into the Server Action |
| read-only role → mutation | auditor/dpo/socio could POST directly to the action |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-11 | Elevation of Privilege | template mutation by read-only role | mitigate | assertNotReadOnly() first in every mutating action + role gate admin/superadmin/ti |
| T-08-12 | Tampering | malformed template content | mitigate | zod documentTemplateSchema validates name/category/content length at action entry |
| T-08-13 | Information Disclosure | cross-tenant template read | mitigate | listTemplates uses createClient() → RLS clinic_id = get_my_tenant_id() |
| T-08-14 | Information Disclosure | internal errors to client | mitigate | actions return Portuguese user-facing messages; never raw DB errors |
</threat_model>
